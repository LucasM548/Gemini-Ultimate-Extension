# Privacy Policy for Gemini Ultimate

**Effective Date:** December 9, 2024

## 1. Data Collection
The "Gemini Ultimate" extension does **not** collect, store, share, or transmit any personally identifiable information (PII) or user data to external servers.

## 2. Permissions and Usage
- **Host Permissions (`https://gemini.google.com/*`)**: Used exclusively to interact with the DOM of the Gemini webpage to automate button clicks and text insertion based on user preferences.
- **Storage**: Used locally `chrome.storage.sync` to save your configuration preferences (e.g., target models, delay settings). This data is synced across your Chrome browsers signed in with the same account but is not accessible to the developer or third parties.

## 3. Remote Code
This extension does not use any remote code. All functionality is contained within the package installed on your browser.

## 4. Contact
If you have questions about this policy, please contact the developer via the GitHub repository issues page.
